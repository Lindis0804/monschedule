# Implementation

1. Clone or download project:

```bash
git clone https://github.com/Lindis0804/monschedule
```

2. Backend:

- Switch to folder backend:

```bash
cd backend
```

<br/>
<div align="center">
<img src="./images/cd_backend.png" style="width:50%">
</div>
<br/>
- Install neccessary packages:

```bash
npm install
```

- Run server:

```bash
npm start
```
